# Password Strength Checker

##  Overview
The project determines password strength by assessing three distinct levels of strength which include weak passwords and medium passwords and strong passwords.

##  Features
The system verifies password length requirements through its first feature which checks password length. The system identifies all four character types which include uppercase letters and lowercase letters and numeric digits and special symbols.

##  How to Run
1. You need to install Python on your computer.
2. Execute the command, "python main.py," to start the program.

##  Example
Input: Abc@12345
Output: Strong
